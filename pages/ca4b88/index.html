EasyMock

# EasyMock 模拟测试辅助工具

开源网址

官网：https://easy-mock.com/ (暂时无法访问)

github：https://github.com/easy-mock/easy-mock

# 简介

Easy Mock 是一个可视化，并且能快速生成模拟数据的持久化服务。

# 特性

  • 支持接口代理
  • 支持快捷键操作
  • 支持协同编辑
  • 支持团队项目
  • 支持 RESTful
  • 支持 Swagger (opens new window) | OpenAPI Specification (1.2 (opens new window) & 2.0 (opens new window) & 3.0 (opens new window))
    • 基于 Swagger 快速创建项目
    • 支持显示接口入参与返回值
    • 支持显示实体类
  • 支持灵活性与扩展性更高的响应式数据开发
  • 支持自定义响应配置（例：status/headers/cookies）
  • 支持 Mock.js (opens new window) 语法
  • 支持 restc (opens new window) 方式的接口预览

# docker-compose.yml

version: '3.7'

services: 

  easymock: 
    image: jptx1234/easy-mock-all-in-one
    container_name: easymock
    restart: always
    ports:
      - 7300:7300
    environment:
      TZ: Asia/Shanghai
    networks: 
      - swarm_net 

networks: 
  swarm_net: 
    external: true

# deploy.sh

#!\bin\bash

docker-compose up -d
